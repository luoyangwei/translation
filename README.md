# VSCode translation 插件

给英语不好的同学使用, 使用快捷键直接翻译并且替换内容.

## 快速翻译

我映射了 MacOS 的`cmd+.`和 windows 的`ctrl+.`键盘快捷键.

![img](https://raw.githubusercontent.com/luoyangwei/translation/master/demo.gif)

## Release Notes
